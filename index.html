How It Works

Entering Stock: Fill in the date (required for all items in a session; must be the same date or clear data to start new), product, units, wholesale, and retail prices. Click "Add Item" to add to the table.

Display: The table updates live with totals, and a summary section shows overall calculations including profit margins.

Download: Click "Download as CSV" to get a spreadsheet file (opens in Excel/Google Sheets). It includes the date, items, and totals.

Upload: Select a CSV file (from a previous download) to load data back into the page. It assumes one date per file.

Clear: Resets everything and starts fresh.
